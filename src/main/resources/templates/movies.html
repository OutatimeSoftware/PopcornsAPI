How to use the movies endpoint:

This is a brief explanation of how to use the movies endpoint /api/movies.
Get all the movies

Using the GET method to /api/movies will return the complete movies list.

[ {"ID": Integer, "title": String, "addedBy": User, "createdAt": Date, "updatedAt": Date}, ... ]
Get a movie

Using the GET method to /api/movies/{id} will return the desired movie.

{"ID": 1, "title": "Soul", "addedBy": {"username": "imreyesjorge", ...}, "createdAt": {"day": 12, ...}, "updatedAt": {"day": 13, ...}}
Create a new movie

Using the POST method to /api/movies will receive an object that looks like this in the body of the HTTP Request

{"ID": Integer, "title": String, "addedBy": User "createdAt": Date, "updatedAt": Date}

and return the freshly created item.

{"ID": 2, "title": "LoTR", "addedBy": {"username": "miguelRAvila", ...}, "createdAt": {"day": 11, ...}, "updatedAt": {"day": 12, ...}}
Update a movie

Using the PUT method to /api/movies/{id} will receive an object that looks like this in the body of the HTTP Request

{"ID": Integer, "title": String, "addedBy": User, "createdAt": Date, "updatedAt": Date}

and return the freshly updated item.

{"ID": 3, "title": "Harry Potter", "addedBy": {"username": "robcllanes", ...}, "createdAt": {"day": 22, ...}, "updatedAt": {"day": 11, ...}}
Delete a movie

Using the DELETE method to /api/movies/{id} will receive an object that looks like this in the body of the HTTP Request

{"ID": Integer, "title": String, "addedBy": User, "createdAt": Date, "updatedAt": Date}

and return the freshly deleted item.

{"ID": 4, "title": "Avatar", "addedBy": {"username": "marioC", ...}, "createdAt": {"day": 30, ...}, "updatedAt": {"day": 31, ...}}
