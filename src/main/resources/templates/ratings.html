How to use the ratings endpoint:

This is a brief explanation of how to use the ratings endpoint /api/ratings.
Get all the ratings

Using the GET method to /api/ratings will return the complete ratings list.

[ {"ID": Integer, "movieTitle": String, "rating": Integer, "review": String}, ... ]
Get a rating

Using the GET method to /api/ratings/{id} will return the desired rating.

{"ID": 1, "movieTitle": "Soul", "rating": 4, "review": "Lorem ipsum dolor at sim..."}
Create a new rating

Using the POST method to /api/ratings will receive an object that looks like this in the body of the HTTP Request

{"ID": Integer, "movieTitle": String, "rating": Integer, "review": String}

and return the freshly created item.

{"ID": 2, "movieTitle": "Avatar", "rating": 5, "review": "Lorem ipsum dolor at sim..."}
Update a rating

Using the PUT method to /api/ratings/{id} will receive an object that looks like this in the body of the HTTP Request

{"ID": Integer, "movieTitle": String, "rating": Integer, "review": String}

and return the freshly updated item.

{"ID": 3, "movieTitle": "Up", "rating": 3, "review": "Lorem ipsum dolor at sim..."}
Delete a rating

Using the DELETE method to /api/ratings/{id} will receive an object that looks like this in the body of the HTTP Request

{"ID": Integer, "movieTitle": String, "rating": Integer, "review": String}

and return the freshly deleted item.

{"ID": 4, "movieTitle": "Wonder Woman", "rating": 2, "review": "Lorem ipsum dolor at sim..."}
