How to use the users endpoint:

This is a brief explanation of how to use the users endpoint /api/users.
Get all the ratings

Using the GET method to /api/users will return the complete users list.

[ {"ID": Long, "username": String, "name": String, "lastname": String, "profilePicture": String, "email": String}, ... ]
Get a user

Using the GET method to /api/users/{id} will return the desired user.

{"ID": 1, "username": "kings", "name": "Jorge", "lastname": "Reyes", "profilePicture": "https://lh3.googleusercontent.com/-oCS2G33ogSU/AAAAAAAAAAI/AAAAAAAAAAA/fntcdzg69Fk/s181-c/116225983760077661019.jpg", "email": "test@reyes.cool"}
Create a new user

Using the POST method to /api/users will receive an object that looks like this in the body of the HTTP Request

{"ID": Long, "username": String, "name": String, "lastname": String, "profilePicture": String, "email": String}

and return the freshly created item.

{"ID": 2, "username": "mariochan", "name": "Mario", "lastname": "Chan", "profilePicture": "https://lh3.googleusercontent.com/-oCS2G33ogSU/AAAAAAAAAAI/AAAAAAAAAAA/fntcdzg69Fk/s181-c/116225983760077661019.jpg", "email": "mario@chan.zurita"}
Update a user

Using the PUT method to /api/users/{id} will receive an object that looks like this in the body of the HTTP Request

{"ID": Long, "username": String, "name": String, "lastname": String, "profilePicture": String, "email": String}

and return the freshly updated item.

{"ID": 3, "username": "miguelravila", "name": "Miguel", "lastname": "Avila", "profilePicture": "https://lh3.googleusercontent.com/-oCS2G33ogSU/AAAAAAAAAAI/AAAAAAAAAAA/fntcdzg69Fk/s181-c/116225983760077661019.jpg", "email": "miguelr@avila.cool"}
Delete a user

Using the DELETE method to /api/users/{id} will receive an object that looks like this in the body of the HTTP Request

{"ID": Long, "username": String, "name": String, "lastname": String, "profilePicture": String, "email": String}

and return the freshly deleted item.
